08_Python函数式编程

1 高阶函数

Python 中的函数是“一等函数”，所谓“一等函数”指的就是函数可以赋值给变量，函数可以作为函数的参数，函数也可以作为函数的返回值。把一个函数作为其他函数的参数或返回值的用法，我们通常称之为“高阶函数”。

在Python中，func()是函数调用，func是函数本身，可以把函数本身赋值给变量，也就是：变量指向函数。

其实函数名本身也是变量，函数名指向函数。

函数能接收变量，变量可以指向函数，所以函数可以接受函数，这时传递的不是数据，而是计算逻辑。

1.1 map()

map()接收两个参数，一个是函数，一个是可迭代对象Iterable，map()将函数依次作用到可迭代对象的每个元素，并将结果作为一个新的迭代器对象Iterator返回。

1.2 reduce()

reduce()把一个函数作用到一个序列中，其中这个函数必须接收两个参数，reduce()将计算结果继续和下一个元素累计计算。 使用reduce()需要导入from functools import reduce。

1.3 filter()

filter接收一个函数一个序列，把传入的函数作用到每一个元素上，根据返回值决定是否保留，True保留元素，False丢弃元素。最终结果作为Iterator返回，也就是一个惰性序列，需要使用list()获取所有计算结果。

1.4 sorted()

sorted()可以对序列进行排序，返回一个新的列表。

还可以接收一个key函数实现自定义排序，key将指定函数作用到序列的每一个值上，并按照返回值进行排序。另外还可以使用reverse=True表示逆序排列。

3 闭包

3.1 闭包的定义

调用了外部函数变量的内部函数就称之为闭包（Closure），闭包可以增强安全性，将原本的全局变量变为外部函数的参数，对于内层函数来说这个变量又是外部变量。

闭包的三个必要条件：

  1. 有嵌套：函数嵌套
  2. 有引用：内部函数引用外部函数变量
  3. 有返回：外部函数返回内部函数名（对象）

银行取款非闭包写法：这种写法余额initial_value是全局变量，有被修改的风险。

闭包写法：initial_value变成了外部函数count()的参数，外部无法访问。想在内部函数引用外部变量需要使用nonlocal声明，否则会被认为是内部函数局部变量。

3.2 闭包的原理

闭包有点类的思想，外层函数中的参数像类的元素一样被保留了下来。实际上在Python的函数对象中存储了一些属性：

  • __code__：保存函数的字节码。
  • __globals__：保存全局作用域引用。
  • __closure__：保存了被捕获的自由变量（也就是闭包变量）。

在上述案例中fn = count(10000)执行后，fn是atm这个函数对象。所以initial_value=10000 这个变量其实是保存在 fn.__closure__ 里的，Python会把它包装为一个cell object（细胞对象），用来存活并保持可变。

4 装饰器

4.1 基本用法

装饰器（Decorator） 就是创建一个闭包函数，在闭包函数内调用目标函数，可以达到不改变目标函数的同时，增加额外的功能。

上述写法还不够优雅，可以使用@函数名的方式，也叫语法糖。

原来的 demo()函数仍然存在，只是同名demo变量指向了新的函数，于是调用demo()会执行扩展后的函数，其实就是保存了 func 状态的 inner()函数。

4.2 带参有返回值

要被装饰的原函数带参有返回值，并且原函数接收不定长参数，有一些细节需要注意。

  • fn(args, kwargs)是位置参数：相当于get_sum((1,2,3,4), {'math':18})
  • fn(*args, **kwargs)是不定长参数：相当于get_sum(1,2,3,4, math=18)

4.3 多个装饰器

多个装饰器一层一层进入函数，再一层一层返回，装饰器的应用顺序是：自下而上，依次包裹。

  • 执行顺序是由外到内（先进入 outer1，再进入 outer2，再执行原函数）
  • 返回顺序是由内到外（先从 outer2 返回，再从 outer2 返回）

上述代码的两个装饰器相当于hello = outer1(outer2(hello))。

  1. hello先被outer2装饰，返回一个新函数
  2. 这个新函数再被outer1装饰，又得到一个新函数

多个装饰器有点递归或者栈调用的感觉，只不过多个装饰器是采用多层函数嵌套，并没有无限调用自己，只是采用了类似的套娃结构。

如果说递归是：从前有座山，山里有座庙，庙里有个老和尚，正在给小和尚讲故事！故事是…… 从前有座山，山里有座庙，庙里有个老和尚，正在给小和尚讲故事！故事是…… 递归是：无限重复的同一个壳。

那么多个装饰器是：从前有座山，山里有座庙，庙里有个老和尚，正在给小和尚讲故事！故事是…… 从前有条河，河里有条船，船上有个老船员，正在给小船员讲故事！故事是…… 多个装饰器是：多层不同的外壳。

4.4 带参装饰器

在普通装饰器中@outer 相当于 demo = outer(demo)，默认接收一个被装饰的函数参数，这里所说的带参装饰器是在@outer的参数，语法糖的格式只允许接收一个参数，所以要使用三层函数嵌套来进行额外的传参。

在上述代码中，@deco('猪猪侠')相当于hello = deco('猪猪侠')(hello)，hello实际是一个指向inner函数的变量。

5 偏函数

Python的functools模块提供了很多有用的功能，其中一个就是偏函数（Partial function）。 偏函数可以固定函数的某些参数，生成一个新的函数，在调用时可以减少重复参数的输入，降低调用难度。
