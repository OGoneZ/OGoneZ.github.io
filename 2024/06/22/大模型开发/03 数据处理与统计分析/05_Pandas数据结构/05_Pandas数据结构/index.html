05_Pandas数据结构

Numpy和Pandas的关系：就像先掌握算术才能学代数一样，NumPy就是数据分析的”算术基础”，虽然可以直接用计算器（Pandas），但理解底层原理才能走得更远。

Pandas 核心数据机构是Series(数据系列)、DataFrame(数据窗/数据框)，Series可以看做表中的一列，DataFrame则是整个表格，由多个列构成。除此之外还有一个特殊的Index类，为Series和DataFrame提供索引功能。

1 Series对象

1.1 创建 Series 对象

Series对象内部结构：一个ndarray用来保存数据，一个index用来保存索引。类似Numpy的一维数组，但是增加了“标签”，具有数据和索引。

2 DataFrame 对象

DataFrame是一个二维的表格型数据结构，既有行索引，也有列索引，可以将其看作由多个Series按列组成的一个字典。是 Pandas 中最常用的数据结构，可以用来保存和处理异质的二维数据，每个列可以是不同的数据类型，这也是和 NumPy 数组的最大区别。

数据表格三要素贯穿 Pandas 数据分析的始终：

  • column：对应着列名，也就是列索引
  • index：代表行索引
  • value：则指代具体的数据值

2.2 数据读取和存储

实际应用中，单独创建数据结构的情况较少，更多的是通过读取外部数据文件来创建数据结构。

2.2.1 读取 Excel

Pandas 默认会将第一行数据作为列名，如果想自定义列名，可以将header=None告诉Pandas数据中没有表头，然后通过names参数指定列名。

通过read_csv函数可以读取 CSV 和 TSV 文件，默认使用UTF-8编码格式，如果是其他编码格式，可以通过encoding参数指定。

2.4.5 apply自定义函数

可以将一个函数应用（apply）到 Series 的每个元素上，或者 DataFrame 的每一行或每一列上。

当作用于 Series（DataFrame 的一列）时，apply() 会将函数依次作用于 Series 中的每一个元素。

0    高
1    低
2    中
Name: 英语, dtype: object

当作用于 DataFrame 时，apply() 可以将函数作用于每一列或每一行。

  • func：要应用的函数
  • axis：指定函数应用方向
    • 0：默认值，将函数应用到每一列，接收对象是Series
    • 1：将函数应用到每一行，接收对象是Series

沿列应用时，函数接收到的一列数据，通常用于聚合（如求和、最大值等）或对整列进行转换。

语文     8
数学    35
英语    38
dtype: int64

沿行应用时，函数接收到的是一行数据，索引为列名，通常用于使用多列计算新值。

1 李四 82 65 52 69.2

3 Index 对象

Index用来唯一标识 DataFrame 中的每一行数据，就像数据库表中的主键一样，但它不强制要求是唯一的，类似于一维数组，存储着用于标记各行数据的标签。

3.1 Index 的特点

  1. 标签：作为行标签使用，用来访问特定行df.loc[行标签]
  2. 对齐：两个DataFrame之间运算（比如加法、合并等）时，Pandas根据Index和Column自动将数据对齐，而不需要行和列的顺序相同
  3. 不可变性：Index对象一旦创建，不允许被修改，所有修改索引必须整体修改
