12_Python面向对象高级

1 __slots__限制属性

Python可以通过 对象.属性的方式很方便的添加属性值，但是这种方式太自由了，Python可以定义一个特殊的变量 __slots__来限制类的属性。

__slots__定义的属性仅对当前类起作用，对继承的子类是不起作用的，除非在子类中也定义 __slots__，这样，子类实例允许定义的属性就是自身的 __slots__加上父类的 __slots__。

2 @property方法当属性

如果直接把属性暴露出去，可以直接修改属性，但是没办法对值的合法性进行检验，所以可以将属性设为私有属性，然后通过 getter方法和 setter方法进行私有属性的获取和设置。

但是调用方法终归是略显复杂，没有直接使用属性简单，作为追求简单的Python语言，内置的 property可以将方法当做属性使用，常用于 getter方法和 setter方法。既能检查参数，又可以用类似属性的简单方式来访问类的变量。

2.1 装饰器方法

  • @property修饰获取值的方法
  • 获取值方法名.setter修饰设置值的方法

通过 @property 装饰器，创建了一个名为 age 的“公共接口”或“属性”。这个属性在外部看起来就像一个普通属性，可以用 p1.age 来访问它，但实际上，这个访问背后调用的是 age() 方法，而赋值时调用的则是 age() 方法下的 age.setter。

3 枚举类

枚举类就是一种实例有限且固定的特殊类，当需要定义一组命名常量时，可以选择枚举类。

枚举类正常情况下允许成员有重复值，如果使用 @unique可以控制枚举成员不能有重复值。
